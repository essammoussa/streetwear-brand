# VOIDWEAR - Streetwear Storefront

A modern, high-performance, minimalist e-commerce storefront for a streetwear brand built with React, Vite, and Tailwind CSS.

---

## 🚀 How to Run the Project / طريقة تشغيل المشروع

### English (EN)

To run this project locally on your machine, follow these steps:

1. **Install Node.js**: Ensure you have Node.js installed on your computer. You can download it from [nodejs.org](https://nodejs.org/).
2. **Open Terminal**: Open your terminal (or Command Prompt / PowerShell) and navigate to the project folder.
3. **Install Dependencies**: Run the following command to install all required packages:
   ```bash
   npm install
   ```
4. **Start Development Server**: Run the following command to start the local server:
   ```bash
   npm run dev
   ```
5. **View Website**: Open your browser and go to `http://localhost:5173` to view the website.(Follow link (ctrl + click))

### العربية (AR)

لتشغيل هذا المشروع على جهازك الشخصي، اتبع الخطوات التالية:

1. **تثبيت Node.js**: تأكد من تحميل وتثبيت برنامج Node.js على جهازك. يمكنك تحميله من [nodejs.org](https://nodejs.org/).
2. **فتح موجه الأوامر (Terminal)**: قم بفتح موجه الأوامر وانتقل إلى المجلد الخاص بالمشروع.
3. **تثبيت الحزم (Dependencies)**: قم بتشغيل الأمر التالي لتحميل جميع الملفات اللازمة لتشغيل المشروع:
   ```bash
   npm install
   ```
4. **تشغيل السيرفر المحلي**: قم بتشغيل الأمر التالي لفتح الموقع:
   ```bash
   npm run dev
   ```
5. **عرض الموقع**: افتح المتصفح الخاص بك وانتقل إلى الرابط `http://localhost:5173` لتصفح الموقع. (Follow link (ctrl + click))

## 🛠️ Tech Stack
- **React 19**
- **Vite**
- **Tailwind CSS** (v3)
- **React Router v6**
- **Lucide React** (Icons)

## ✨ Features
- Monochromatic / Minimalist design system.
- Full E-Commerce layout (Home, Products Grid, Product Details).
- Interactive Checkout flow.
- Functional global Cart (Context API).
- Size Guide modal & Account Dropdown.
- Fully responsive design (Mobile, Tablet, Desktop).
